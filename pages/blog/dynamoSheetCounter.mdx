# Dynamo sheet counter
In this video I describe a dynamo definition that counts the total number of sheets in a submission set and sets the appropriate parameters in the sheet.  The main take away from this is how to Sort Revit Elements by Parameter Value. This is a major help in culling elements to then do further work.
